# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

A collection of Python learning examples organized by topic. Not a single application — each `.py` file is a standalone script.

## Repository Structure

- **python-base/** — Core Python tutorials organized in numbered sections:
  - `01-Basics/` — Data types, strings, sequences, dictionaries, statements, modules
  - `02-OOP/` — Classes, inheritance, polymorphism, iterators, static methods, properties
  - `03-Builtins/` — Standard library usage (sys, webbrowser, module inspection)
  - `04-Web/` — Socket programming, WSGI servers, basic Flask
  - `05-Tests/` — unittest examples
  - `06-Extending/` — Jython/Java interop
- **flask/** — Flask web app examples (routing, request/response, forms, SQLAlchemy, templates)

## Commands

### Run Flask examples
```
python flask/HelloFlask.py
```
